# Crypto API Data Collection

This project collects cryptocurrency data using the CoinMarketCap API and stores it for analysis.

## Tools Used
- Python
- Pandas
- Requests
- Jupyter Notebook

## Features
- Fetch real-time crypto data
- Convert JSON data to Pandas DataFrame
- Automate API calls for continuous data collection

## File
- crypto_api_data_collection.ipynb : Main notebook
